-Inventory Management System
An Inventory Management System built using .NET Core API, React.js, Tailwind CSS, and JSON-based data storage (no database).
This system allows managing Categories, Products, Sorting, Searching, and CRUD Operations with a clean UI and reusable components.

-Tech Stack
  Backend
    .NET Core Web API 8.0
  
  Frontend
    React.js (Vite)
    Tailwind CSS
    Axios

-Features
  Category & Product Management
  Add / Edit / Delete
  Search and Filter Products
  Sort by Price / Stock
  Pagination for products
  Modern UI with Tailwind CSS
  Toast and Modal Alerts
  Logging support

-Setup Instructions
1. Clone the Project
2. Install Dependencies for Backend (dotnet restore)
3. Install Dependencies for Frontend (npm install)
4. Run both

-Screenshots of final UI
Placed in frontend public folder

-Limitations
Can't handle subcategories, categories only.
JSON File Storage (Not Scalable)
No Authentication or Authorization
No Image upload for products
