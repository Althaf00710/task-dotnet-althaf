-Inventory Management System
An Inventory Management System built using .NET Core API, React.js, Tailwind CSS, and JSON-based data storage (no database).
This system allows managing Categories, Products, Sorting, Searching, and CRUD Operations with a clean UI and reusable components.

-Tech Stack
1. Backend
    .NET Core Web API 8.0
  
2. Frontend
    React.js (Vite),
    Tailwind CSS,
    Axios

-Features
1. Category & Product Management
2. Add / Edit / Delete
3. Search and Filter Products
4. Sort by Price / Stock
5. Pagination for products
6. Modern UI with Tailwind CSS
7. Toast and Modal Alerts
8. Logging support

-Setup Instructions
1. Clone the Project
2. Install Dependencies for Frontend (npm install)
3. Run both

-Screenshots of final UI
Placed in frontend public folder

-Limitations
1. Can't handle subcategories, categories only.
2. JSON File Storage (Not Scalable)
3. No Authentication or Authorization
4. No Image upload for products
